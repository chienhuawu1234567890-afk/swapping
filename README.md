# Exact Preservation of Contingency-Table Inference through Category-Preserving Label Swaps

This repository contains the LaTeX manuscript and supporting files for the paper:

**Exact Preservation of Contingency-Table Inference through Category-Preserving Label Swaps**

## Overview

This manuscript proposes a category-preserving label-swapping mechanism for protecting sensitive subject-level group labels in categorical microdata while preserving contingency-table inference exactly.

The proposed mechanism exchanges group labels only among subjects within the same outcome category. Therefore, the group-by-category contingency table remains unchanged after disclosure protection. As a result, inferential procedures based solely on contingency-table counts are preserved exactly, including Pearson's chi-square tests, likelihood-ratio tests, Fisher's exact tests, and related deterministic functions of the contingency table.
